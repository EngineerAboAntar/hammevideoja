## ✨ 主な機能

* 高速な動画ダウンロード
* 複数のプラットフォームに対応
* シンプルで使いやすいユーザーインターフェース
* ダウンロード前に画質を選択可能
* 効率的なダウンロード管理機能
* 複数ファイルの同時ダウンロード対応
* 低いシステムリソース消費
* Windows向けに最適化されたモダンなデザイン

#

## 🖥️ 動作環境

* Windows 10 以降
* インターネット接続

## 🚀 使い方

1. 動画のURLをコピーします。
2. HammeVideoを起動します。
3. URLを入力欄に貼り付けます。
4. 希望する画質を選択します。
5. ダウンロードボタンをクリックします。
6. ダウンロード完了までお待ちください。
